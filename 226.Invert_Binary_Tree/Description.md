# 226. Invert Binary Tree
## Link
[Invert Binary Tree](https://leetcode.com/problems/invert-binary-tree/description/)
## Description
Invert a binary tree.
```C++
     4
   /   \
  2     7
 / \   / \
1   3 6   9
```
to
```C++
     4
   /   \
  7     2
 / \   / \
9   6 3   1
```
